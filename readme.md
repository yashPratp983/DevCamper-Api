# DevCamper API

> Backend API for the DevCamper application, which is a bootcamp directory website

## Usage

Rename "config/config.env.env" to "config/config.env" and update the values/settings to your own

## Install Dependencies

```bash
npm install
```

## Run App

```bash
# Run in dev mode
npm run dev

# Run in prod mode
npm start
```
